1. circle객체 생성

- 속성 : rad(반지름)
- 메서드 : 면적구하기(getArea()) (PI*R^2), 둘레구하기 (getCircum())(2*PI*R)



2. Rect객체 생성

- 속성 : 가로(width), 세로(height), 이름(name)
- 메서드 : 면적구하기(getArea()), 둘레구하기 (getCircum())



3. addr객체

- 정해진 속성과 메소드가 없다 var addr={ }
- 확인버튼을 클릭하면 동적으로 속성과 메소드를 추가해서 사용할 수 있다.
addr.name = "홍길동" ...
